# Student Skill Tracker
A Java Web Application developed using JSP, Servlets, JDBC, and MYSQL.

## Features
- User Login
- Dashboard
- Skills Management
- Categories Management
- Student Management
- Assign Skills
- Update Skill Progress
- Reports

## Technologies 
- Java
- JSP & Servlet
- JDBC
- MYSQL
- HTML
- CSS
- Apache Tomcat
